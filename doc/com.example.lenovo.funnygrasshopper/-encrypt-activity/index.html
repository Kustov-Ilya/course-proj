EncryptActivity

class EncryptActivity : Any

Окно приложения, в котором проиходит шифрование текста в изображение (в случае, если пользователь добавляет его из галереи) или в текст формата Base64 (если изображение не было добавлено). Текст в формате Base64 можно скопировать нажатием на него. Изображение сохраняется нажатием на него. Сохраняет в папку GrassHopper на SD в формате PNG
